# HNXS Font Manager

Windows에서 폰트 파일을 빠르게 확인하고 정리하는 폰트 관리 프로그램입니다.

폴더나 ZIP 파일 안에 흩어져 있는 폰트를 불러와 설치 여부, 중복 가능성, 버전, 형식, 파일명을 한 화면에서 확인할 수 있습니다. 필요한 폰트만 선택해 설치하고, 폰트 이름을 기준으로 파일명을 일괄 정리할 수 있도록 만들었습니다.

## 핵심 기능

### 폴더와 ZIP 파일 불러오기

HNXS Font Manager는 폴더 또는 ZIP 파일을 지정해 폰트 파일을 가져옵니다.

지원 형식:

- `.ttf`
- `.otf`
- `.ttc`

ZIP 파일은 임시 작업 공간에 풀어서 처리하며, 변경사항이 있을 때 다시 ZIP 파일로 저장할 수 있습니다.

### 빠른 폰트 목록 확인

불러온 폰트는 리스트에서 바로 확인할 수 있습니다.

표시 정보:

- 폰트명
- 파일명
- 버전
- 형식
- Windows 설치 여부
- 중복 후보 여부

검색은 폰트명과 파일명 모두를 기준으로 동작합니다.

### 설치 여부 필터링

Windows에 이미 설치된 폰트와 아직 설치되지 않은 폰트를 구분해서 볼 수 있습니다.

필터:

- 모든 폰트
- 미설치
- 설치됨
- 중복 후보

### 중복 폰트 검사

같은 폰트로 의심되는 항목을 찾아 중복 후보로 표시합니다.

중복 후보만 따로 모아 확인할 수 있으며, 삭제 후보 자동 선택 기능으로 정리 작업을 빠르게 진행할 수 있습니다.

### 선택 설치

체크박스로 필요한 폰트만 선택해 설치할 수 있습니다.

설치 방식:

- 현재 사용자 설치
- 시스템 전체 설치

일반 실행 상태에서는 현재 사용자 설치가 기본값입니다. 관리자 권한으로 실행하면 시스템 전체 설치가 기본값으로 설정됩니다.

### 파일명 일괄 변경

폰트 내부의 이름 정보를 읽어 파일명에 반영할 수 있습니다.

파일명으로 사용할 수 없는 문자는 Windows 파일명 규칙에 맞게 자동 변환합니다. 여러 폰트를 한 번에 정리할 때 유용합니다.

### 다국어 폰트 미리보기

영문, 한글, 일본어, 한자, 기호가 포함된 기본 문구로 폰트 표시 상태를 확인할 수 있습니다.

폰트에 한글 글리프가 없는 경우에는 미리보기에서 시스템 폰트로 대체될 수 있으며, 프로그램에서 이를 안내합니다.

### 이전 작업 자동 복원

프로그램을 다시 실행하면 마지막으로 열었던 폴더 또는 ZIP 파일을 자동으로 다시 불러옵니다.

폴더를 다시 열 때는 모든 파일을 처음부터 다시 읽지 않고, 추가되거나 변경된 파일만 다시 분석해 속도를 높입니다.

## 속도 중심 설계

HNXS Font Manager는 Windows 환경에서 많은 폰트를 빠르게 다루는 것을 가장 중요한 목표로 합니다.

이를 위해 폴더 스캔 시 파일 경로, 크기, 수정 시간을 캐시해 변경되지 않은 폰트는 이전 분석 결과를 재사용합니다. 새로 추가되었거나 변경된 파일만 다시 읽기 때문에 재실행 후 목록 확인이 빠릅니다.

## 사용 대상

HNXS Font Manager는 다음과 같은 사용자에게 적합합니다.

- 프로젝트별 폰트를 따로 관리하는 디자이너
- 클라이언트나 브랜드 작업용 폰트를 자주 확인해야 하는 사용자
- ZIP으로 받은 폰트 패키지를 설치 전 검토해야 하는 사용자
- 중복 폰트와 불규칙한 파일명을 정리하고 싶은 사용자
- Windows에서 빠르게 동작하는 실용적인 폰트 관리 도구가 필요한 사용자

## 시스템 요구사항

- Windows 10 또는 Windows 11
- 배포용 실행 파일 사용 시 별도 .NET 설치 불필요
- 시스템 전체 설치 기능 사용 시 관리자 권한 필요

## 현재 버전

`v0.1.0`

## 주의사항

- 시스템 전체 폰트 설치는 Windows 관리자 권한이 필요합니다.
- 일부 폰트는 한글, 일본어, 한자 글리프를 포함하지 않을 수 있습니다.
- ZIP 파일 작업 시 변경사항을 저장해야 압축 파일에 반영됩니다.
- 폰트 라이선스는 폰트 제작사 또는 배포처의 조건을 확인해야 합니다.

## 한 줄 소개

HNXS Font Manager는 폰트 파일을 빠르게 불러오고, 설치 여부와 중복 상태를 확인하며, 선택 설치와 파일명 정리를 한 번에 처리하는 Windows용 폰트 관리 프로그램입니다.
