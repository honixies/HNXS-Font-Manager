HNXS Font Manager v0.1.0
========================

Install for current user:
  Right-click Install-HNXS-Font-Manager.ps1 and choose "Run with PowerShell"

Install for all users:
  Open PowerShell as Administrator, then run:
    .\Install-HNXS-Font-Manager.ps1 -AllUsers

Uninstall:
  Use the Start Menu shortcut "Uninstall HNXS Font Manager"
  or run:
    .\Uninstall-HNXS-Font-Manager.ps1

Notes:
- The app is packaged with its .NET runtime dependencies.
- System-wide font installation inside the app still requires administrator privileges.
